About Us

TrackHive is a real-time classroom occupancy monitoring system designed to improve energy efficiency and classroom management. Using dual sonar sensors at classroom entrances, our system counts the number of people entering or leaving. This information is securely stored using the Blynk Web API and displayed through our web app interface. When occupancy reaches zero, TrackHive automatically switches off classroom electronics, reducing unnecessary power consumption and ensuring sustainable use of resources.

Real-Time Monitoring

Accurately tracks classroom occupancy using dual sonar sensors placed at entrances.

Energy Savings

Automatically switches off lights, fans, and other electronics when classrooms are empty.

Secure Data

Occupancy data is updated and stored through Blynk’s secure web API for reliability and safety.

System Transparency

Teachers and administrators can view real-time electronic status and classroom usage.

Future Development

Planned expansion into public transport occupancy monitoring, including train compartments and buses.
